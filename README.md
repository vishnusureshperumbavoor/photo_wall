# Photo Wall

## Hosting
[https://trenser-afb97.web.app/](https://trenser-afb97.web.app/)

## Prerequisits

- NodeJS
- Visual Studio Code

## Technologies

- ReactJS (Backend)
- React Redux
- Firebase (Database)

## Features
- Add photos
- Add comments
- Delete photos

## How to set up

Clone the repository

```sh
git clone https://github.com/vishnusureshperumbavoor/photo_wall.git
code photo_wall
```

Open terminal

```sh
npm install
npm start
```

Now you can use the application at [localhost:3000](https://localhost:3001)
