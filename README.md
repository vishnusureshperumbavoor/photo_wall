# Photo Wall

## Prerequisits

- NodeJS
- Visual Studio Code

## Technologies

- ReactJS (Backend)
- React Redux
- Firebase (Database)

## How to set up

Clone the repository

```sh
git clone https://github.com/vishnusureshperumbavoor/photo_wall.git
code photo_wall
```

Open terminal

```sh
npm install
npm start
```

Now you can use the application at [localhost:3000](https://localhost:3001)
